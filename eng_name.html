인스타그램 이름 추천

영문 이름으로 만들기

1. 20글자 이하의 영문 이름을 입력하세요.

2. 10글자 이하의 영문 성을 입력하세요. (선택)

3. Generate 버튼을 눌러 아이디를 추천 받으세요.

4. 원하는 아이디 옆 Copy 버튼을 눌러 복사하세요!

Generate를 누를 때 마다 다른 결과가 나와요!

이름을 분리해서 입력하면 다른 결과가 나와요!

ex.chulsoo -> chul soo
